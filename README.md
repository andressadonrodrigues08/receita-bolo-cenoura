# 🍰 Receita de Bolo de Cenoura

Projeto desenvolvido para praticar HTML e CSS.

## 💻 Tecnologias
- HTML5
- CSS3

## 🎯 Objetivo
Criar uma página de receita com layout organizado e estilizado.

## ✨ Funcionalidades
- Lista de ingredientes com ícones
- Passo a passo estilizado
- Efeitos de hover
- Imagem central com destaque

## 📚 Aprendizados
- Estruturação com HTML
- Estilização com CSS
- Organização de layout
